# DHT11 Sensor With LCD
DHT11 Temperature And Humidity Sensor With LCD.

## Prerequisites
1. Arduino Uno R3
2. Arduino IDE
3. DHT11 Sensor
4. Breadboard / PCB
5. 4.7k Resistor
6. Jumper Wires
